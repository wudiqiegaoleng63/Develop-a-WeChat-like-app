## <font style="color:#333333;">该项目基础要求</font>
### 前端（Vue）相关基础
#### 1. 前端基础三件套
+ **HTML**：扎实掌握各类 HTML 标签，能够搭建出合理的网页结构框架。像利用 `<header>`、`<nav>`、`<main>`、`<footer>` 等标签进行页面布局，清楚表单元素的不同使用场景，为实现用户交互功能奠定基础。
+ **CSS**：熟练运用选择器精准匹配元素并添加样式，深入理解盒模型原理，能够运用浮动、定位、`flexbox` 和 `grid` 等布局技术，打造出美观且响应式的页面。
+ **JavaScript**：精通 JavaScript 基本语法，熟悉面向对象编程思想，掌握闭包、原型链等高级特性。能够熟练进行 DOM 操作和事件处理，实现丰富的页面交互效果。

#### 2. Vue 相关知识
+ **Vue 核心特性**：透彻理解 Vue 的响应式原理、组件化开发理念，熟练使用各类指令进行数据绑定和逻辑控制。能够独立使用 Vue CLI 创建项目，理解项目的目录结构和配置文件作用。
+ **Vue 生态系统**：掌握 Vue Router 实现单页面应用的路由管理，包括路由导航、路由守卫等功能；了解 Vuex 的状态管理机制，能在大型项目中合理运用其进行数据共享和状态更新。

### 后端（Go）相关基础
#### 1. Go 语言基础
+ **Go 语法与特性**：熟悉 Go 语言的基本语法结构，如变量声明、数据类型、控制流语句等；掌握 Go 的并发编程模型，能够使用 `goroutine` 和 `channel` 高效处理并发任务；了解结构体、接口等面向对象编程概念在 Go 中的实现方式。
+ **标准库运用**：熟练使用 Go 标准库中的 `net/http` 包构建 HTTP 服务器，处理各种 HTTP 请求和响应；掌握 `encoding/json` 包进行 JSON 数据的编解码操作；能够运用 `database/sql` 包与数据库进行交互。

#### 2. 后端开发知识
+ **HTTP 协议原理**：深入理解 HTTP 协议的工作机制，包括请求方法、请求头、响应状态码等；熟悉 RESTful API 设计原则，能够设计出合理、规范的接口。
+ **数据库操作**：掌握关系型数据库（如 MySQL）的基本操作，包括数据库的创建、表的设计、数据的增删改查等；了解数据库的事务处理和索引优化等技术。

### 其他技术基础
#### 1. Redis 基础
+ **数据结构与操作**：了解 Redis 支持的常见数据结构，如字符串、哈希、列表、集合和有序集合，并掌握相应的操作命令。
+ **Redis 应用场景**：熟悉 Redis 在高并发场景下的应用，如缓存穿透、缓存雪崩的处理方法，能够根据实际需求进行合理配置。

#### 2. Kafka 基础
+ **消息队列原理**：理解消息队列的基本概念和作用，了解 Kafka 的架构设计和工作原理，包括主题、分区、副本等核心概念。
+ **Kafka 操作**：掌握 Kafka 的基本操作，如创建主题、发送和消费消息等；了解 Kafka 的生产者和消费者 API，能够使用 Go 语言编写 Kafka 生产者和消费者代码。

#### 3. WebRTC 基础
+ **实时通信原理**：了解 WebRTC 的基本原理和架构，包括媒体流采集、编解码、传输等过程；熟悉 WebRTC 的 API 接口，如 `getUserMedia`、`RTCPeerConnection` 等。
+ **信令交互**：理解 WebRTC 中信令交互的重要性，掌握常见的信令协议和实现方式，能够实现基于 WebRTC 的简单音视频通话功能。

### 通用基础
#### 1. 版本控制
熟练掌握 Git 的基本操作，包括仓库的创建、分支管理、代码提交、合并和冲突解决等；能够使用 GitHub、GitLab 等代码托管平台进行项目的版本管理和团队协作。

#### 2. 网络与服务器知识
了解计算机网络的基本原理，如 TCP/IP 协议栈、端口号、防火墙等；熟悉 Linux 操作系统的基本命令和操作，能够进行服务器的部署、配置和维护。

#### 3. 数据格式与编码
熟练掌握 JSON 数据格式，能够进行 JSON 数据的生成、解析和验证；了解 Base64 编码等常见的数据编码方式，在处理文件和二进制数据时能够灵活运用。
